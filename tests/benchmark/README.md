## 待实现的测试

### CoreMark 基准测试

- **目标**: 测量 IPC (Instructions Per Cycle)
- **预期结果**: IPC > 1.20 (Phase 6 目标)
- **测量内容**:
  - 总周期数
  - 总指令数
  - IPC = 指令数 / 周期数

### Cache 性能测试

- **目标**: 测量 Cache 命中率
- **测量内容**:
  - I-Cache hit/miss 计数
  - D-Cache hit/miss 计数
  - 总体 hit rate = hits / (hits + misses)

### 分支预测性能测试

- **目标**: 测量分支预测准确率
- **测量内容**:
  - 预测次数
  - 错误预测次数
  - 准确率 = (预测次数 - 错误) / 预测次数
