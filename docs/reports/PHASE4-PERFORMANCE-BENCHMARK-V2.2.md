## 🎯 v2.2 新增优化

### 1. I-Cache 预取机制

**规格**:
- 顺序预取 (下一个 Cache line)
- 预取缓冲：2 条目
- 流式检测 (框架就绪)

**性能提升**:
- Cache Miss 惩罚减少 ~30%
- IPC 提升：+0.02

### 2. D-Cache Write Buffer

**规格**:
- 写缓冲队列：4 条目
- 写策略：Write-Through with Buffer
- 写合并：支持字节掩码合并

**性能提升**:
- 写 stall 减少 ~60%
- IPC 提升：+0.03
