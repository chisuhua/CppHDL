## ✅ 今日完成

### 1. 指令译码器 (`include/riscv/rv32i_decoder.h`)

**功能**:
- 32 位指令解析
- 5 种指令格式支持 (R/I/S/B/U/J)
- 控制信号生成
- 立即数符号扩展

**代码行数**: ~150 行

**控制信号**:
| 信号 | 说明 |
|------|------|
| reg_write | 寄存器写使能 |
| alu_src | ALU 源选择 |
| mem_read/write | 内存读写 |
| branch/jump | 分支/跳转 |
| alu_op | ALU 操作码 |

---

### 2. 程序计数器 (`include/riscv/rv32i_pc.h`)

**组件**:
- `ProgramCounter`: PC 寄存器 + 分支选择
- `BranchTargetCalc`: 分支目标计算

**功能**:
- PC 更新 (PC+4 或分支目标)
- 暂停 (stall) 支持
- 复位 (reset) 支持

**代码行数**: ~60 行

---

### 3. 测试框架 (`examples/riscv/rv32i_test.cpp`)

**测试内容**:
- 寄存器文件读写测试
- ALU 运算测试 (ADD/SUB/AND)
- PC 更新测试

**代码行数**: ~180 行
