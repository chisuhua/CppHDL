## 🔧 技术规格

### RV32I 核心

| 特性 | 规格 |
|------|------|
| **指令集** | RV32I (基础整数) |
| **寄存器** | 32×32-bit (x0-x31) |
| **流水线** | 简化 5 级 (IF/ID/EX/MEM/WB) |
| **频率目标** | 25 MHz |
| **指令延迟** | 1-3 周期 |

### 存储器映射

| 地址范围 | 设备 | 大小 |
|---------|------|------|
| 0x0000_0000 - 0x0000_FFFF | SRAM | 64KB |
| 0x4000_0000 - 0x4000_0FFF | GPIO | 4KB |
| 0x4000_1000 - 0x4000_1FFF | UART | 4KB |
| 0x4000_2000 - 0x4000_2FFF | Timer | 4KB |

### AXI4 互联

| 主设备 | 从设备 | 协议 |
|--------|--------|------|
| RV32I Core | SRAM | AXI4-Lite |
| RV32I Core | GPIO | AXI4-Lite |
| RV32I Core | UART | AXI4-Lite |
| RV32I Core | Timer | AXI4-Lite |
